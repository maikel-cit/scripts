# Give execution permission to files

```
chmod +x ./*.sh
```

# Uninstall previous installations

If you have cloned this repo and given execution permissions, run from the repo folder:
```
./purge-docker.sh
```

# Install docker
If you have cloned this repo and given execution permissions, run from the repo folder:
```
./install-docker.sh
```

# Install lando
If you have cloned this repo and given execution permissions, run from the repo folder:
```
./install-lando.sh
```
